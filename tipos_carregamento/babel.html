Babel & Browserslist

Configuração (Browserslist): "Apenas navegadores idosos"

Código Moderno (O que você escreve):

const dobro = (n) => n * 2;

Código Traduzido pelo Babel (O que o idoso entende):

var dobro = function dobro(n) {
  return n * 2;
};
Explicação: O Babel leu a "lista de convidados" (Browserslist), viu que tinha idosos na festa e trocou a Arrow Function (=>) por uma Function comum.
